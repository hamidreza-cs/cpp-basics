# C++ Fundamentals

👋 **About This Repository**

This repository documents my journey learning C++ programming fundamentals as a Computer Science student. It includes exercises, examples, and small console-based projects.

**Level:** Beginner

---

## 📚 Topics Covered

- Variables, Data Types & Operators
- Control Structures (`if`, `switch`, loops)
- Functions & Recursion
- Arrays (1D and 2D)
- Basic Problem Solving & Algorithms
- Clean Code & Commenting

---

## 🚀 How to Run

- Open the project in **VS Code**
- Use `Ctrl + Alt + N` (with Code Runner) for quick execution
- Or `Ctrl + Shift + B` to build
